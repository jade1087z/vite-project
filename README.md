# vite를 이용한 포트폴리오 사이트 만들기

## vite를 사용하는 이유

1.  느렸던 소스 코드 갱신
    기존의 번들러 기반으로 개발을 진행할 때, 소스 코드를 업데이트 하게 되면 번들링 과정을 다시 거쳐야 했었습니다. 따라서 서비스가 커질수록 소스 코드 갱신 시간 또한 선형적으로 증가하게 됩니다.

## 구현

1. 폰트 적용

1. [vite](https://ko.vitejs.dev/guide/)
